2 - abrir o terminal no vscode e digitar: npm init
3 - ele vai pedir as informações do jason nome do chat nome do criador licença etc...
4 - criar um arquivo index.js
5 - ir em scripts no pachage.jason e adicionar esse codigo: "dev": "node index.js" e toda vez para iniciar o projeto digita no terminal: npm run dev ( node pachage manager inicie o script dev)
